# Ftsig Portofolio Manga

## Project Overview
This project serves as a segment of the Ftsig portfolio, showcasing manga-related works and projects.

## Features
- User-friendly interface for browsing manga collections
- Comprehensive details for each manga entry
- Search functionality to easily find desired titles

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ftsig-Git/Ftsig-Portofolio-Manga.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Ftsig-Portofolio-Manga
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
- Start the application:
  ```bash
  npm start
  ```
- Open your web browser and go to `http://localhost:3000` to view the application.
